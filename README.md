# Cover Genius AI

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个基于AI技术的智能封面生成器，能够根据用户输入自动生成精美的社交媒体封面。

## 功能特点

- 🎨 智能生成：基于LLM，自动生成符合需求的封面设计
- 🎯 多平台支持：适配各大主流社交媒体平台的封面尺寸
- 💫 实时预览：即时查看生成效果，所见即所得
